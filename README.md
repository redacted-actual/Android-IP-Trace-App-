Android IP Trace App 

Intuitive IP Tracer for Android

A user-friendly app that lets you enter any public IP address (or trace your own) and instantly view its geolocation details, country, city, ISP, coordinates, and more.Built for curiosity, network testing, or education. Uses the free ip-api.com API, no keys, no tracking, fully ethical and public data only. 

Why This App?

Quick insights into IP origins without complex tools.
Intuitive design: Simple input, clear results, one-tap map view.
Runs on-device with minimal permissions, just internet.

FeaturesAuto-trace your IP on launch for instant results.
Manual IP entry with validation.
Rich details: Country, city, region, zip, latitude/longitude, timezone, ISP, organization.
Map integration: Tap to open location in Google Maps.
Error handling: Friendly messages for invalid IPs or network issues.
Material 3 UI: Modern, responsive, dark/light themes.

Tech StackKotlin + Jetpack Compose — Declarative, beautiful UI.
Retrofit + Gson — Efficient API calls.
Coroutines — Async networking.
Android Intents — For map launching.
